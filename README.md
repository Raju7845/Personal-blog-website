# Rajesh – Personal Portfolio & Blog

This repository contains my personal portfolio and blog website built using Astro.

The website showcases my projects, blog posts, and experiences in data science, artificial intelligence, and data analytics. It serves as a space where I share my work, experiments, and thoughts on technology.

## About the Website

This site highlights:

* My data science and AI projects
* Blog posts about machine learning, AI, and data-related topics
* Portfolio showcasing my work and experiments
* Personal insights and learning journey in technology

## Projects Featured

Some projects included in this portfolio:

* **FinRAG — Indian Mutual Fund Intelligence**
  A Retrieval-Augmented Generation (RAG) system that allows natural language queries over Indian mutual fund datasets.

* **MS Dhoni ODI Performance Dashboard**
  A data visualization project analyzing MS Dhoni’s ODI performance using interactive dashboards.

* **Zepto Inventory & Product Sales Analysis**
  A Power BI dashboard analyzing sales performance, product categories, and discount strategies.

## Tech Stack

This website is built using:

* Astro
* HTML / CSS / JavaScript
* Markdown for blog posts

This project serves as my personal portfolio and blogging platform where I document my learning journey, share insights, and showcase my work in data science and AI.
